# BLOOM

BLOOM is an interactive guide for specialty coffee brewing.

Built to make renowned brewing methods more accessible, it combines recipe customization, visual guidance, brewing timers, and real-time instructions to help users prepare better coffee with consistency and confidence.

Current features include:

* Multiple brewing methods
* Customizable brew ratios and volumes
* Guided brewing workflow with timers
* Step-by-step visual instructions
* Multilingual support
* Post-brew feedback and recipe refinement

This project is still evolving and serves as both a learning tool and an experiment in translating brewing knowledge into a more intuitive digital experience.

If you're a coffee enthusiast and decide to give it a try, I'd genuinely love to hear your thoughts. Feedback, brewing suggestions, bug reports, and feature ideas are always welcome.
