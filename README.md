# BLOOM

BLOOM is an interactive guide for specialty coffee brewing.

Built to make renowned brewing methods more accessible, it combines recipe customization, visual guidance, brewing timers, and real-time instructions to help users prepare better coffee with consistency and confidence.

Currently includes:
- Tetsu Kasuya's 4:6 Method
- James Hoffmann's French Press Method
- Multilingual support (PT, EN, ES)
- Adjustable brew ratios and volumes
- Guided brewing workflow with timers
- Post-brew feedback and recipe adjustment suggestions
